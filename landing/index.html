In 5 Minuten startklar

TradingAgents lokal betreiben

Klonen, Umgebung anlegen, Provider wählen, loslegen. Kein Account, kein SaaS – der komplette Multi-Agenten-Stack läuft auf deiner Maschine.

1

Repository klonen

Hol dir den Quellcode und wechsle ins Verzeichnis.

2

Umgebung anlegen

Python ≥ 3.10 in conda, venv oder deinem Manager der Wahl, dann pip install .

3

Provider konfigurieren

Setze den API-Key deines Providers – oder wähle ollama für einen vollständig offline laufenden Durchlauf ohne Kosten.

4

Analyse starten

Starte die interaktive CLI mit tradingagents und wähle Ticker, Datum, Modell und Research-Tiefe.

5

Oder per Docker

Ein docker compose run genügt – inklusive --profile ollama für den komplett lokalen Betrieb.

# 1 · Klonen
$ git clone https://github.com/TauricResearch/TradingAgents.git
$ cd TradingAgents

# 2 · Umgebung + Installation
$ conda create -n tradingagents python=3.12
$ conda activate tradingagents
$ pip install .

# 3 · Key setzen (oder Ollama für offline)
$ export ANTHROPIC_API_KEY=...

# 4 · Interaktive CLI starten
$ tradingagents
# .env mit deinen Keys befüllen
$ cp .env.example .env

# Standard-Durchlauf
$ docker compose run --rm tradingagents

# Vollständig lokal mit Ollama
$ docker compose --profile ollama \
    run --rm tradingagents-ollama
# Direkt aus deinem Code
from tradingagents.graph.trading_graph import TradingAgentsGraph
from tradingagents.default_config import DEFAULT_CONFIG

config = DEFAULT_CONFIG.copy()
config["llm_provider"] = "ollama"
config["deep_think_llm"] = "qwen3"

ta = TradingAgentsGraph(config=config)
_, decision = ta.propagate("NVDA", "2026-01-15")
print(decision)
Das Team

Ein Handelsdesk aus spezialisierten Agenten

TradingAgents zerlegt die Analyse in klar abgegrenzte Rollen. Jeder Agent hat eigene Werkzeuge und Constraints – gemeinsam bewerten sie den Markt und diskutieren die optimale Strategie.

📊
Analyst-Team

Fundamentals-Analyst

Bewertet Kennzahlen und Bilanzen, ermittelt den inneren Wert und deckt potenzielle Warnsignale auf.

💬
Analyst-Team

Sentiment-Analyst

Verdichtet News-Schlagzeilen, StockTwits und Reddit zu einem Stimmungsbild für die kurzfristige Marktlage.

🌐
Analyst-Team

News-Analyst

Beobachtet globale Nachrichten und Makro-Indikatoren und deutet ihre Wirkung auf die Marktbedingungen.

📈
Analyst-Team

Technical-Analyst

Nutzt Indikatoren wie MACD und RSI, um Muster zu erkennen und Kursbewegungen zu prognostizieren.

⚖️
Researcher-Team

Bull- & Bear-Researcher

Prüfen die Analysen kritisch und wägen in strukturierten Debatten Chancen gegen Risiken ab.

🎯
Execution

Trader-Agent

Fasst alle Berichte zusammen und entscheidet über Zeitpunkt und Größe eines Trades.

🛡️
Risk & Portfolio

Risk-Management-Team

Bewertet Volatilität, Liquidität und weitere Risikofaktoren und passt die Strategie laufend an.

🏦
Risk & Portfolio

Portfolio-Manager

Trifft die finale Entscheidung: Er genehmigt oder verwirft den Handelsvorschlag und gibt die Order frei.

🔁
Gedächtnis

Reflexion & Lernen

Jeder Lauf hält seine Entscheidung fest und speist die realisierte Rendite als Lektion in künftige Analysen ein.

🗺️ Der vollständige Workflow: von den Analysten über die Bull/Bear-Debatte bis zur finalen Freigabe durch den Portfolio-Manager.
Warum lokal

Volle Kontrolle über Daten, Modelle und Kosten

Gebaut mit LangGraph für Modularität – und darauf ausgelegt, unter deiner Kontrolle zu laufen.

🖥️

100 % offline mit Ollama

Über das Ollama-Profil laufen alle Modelle lokal – ohne API-Kosten und ohne dass Daten deine Maschine verlassen.

🔑

Bring your own Key

Auto-Detection der API-Keys per TRADINGAGENTS_*-Env-Vars. Provider wechselst du rein über die .env.

🧠

Persistentes Gedächtnis

Ein Entscheidungs-Log unter ~/.tradingagents reflektiert vergangene Trades und verbessert künftige Läufe.

⏯️

Checkpoint-Resume

LangGraph sichert den Zustand nach jedem Schritt – abgebrochene Läufe setzen genau dort wieder auf, statt neu zu starten.

🐳

Docker-ready

Reproduzierbare Umgebung per docker compose, inklusive eigenem Profil für den vollständig lokalen Ollama-Betrieb.

🌍
